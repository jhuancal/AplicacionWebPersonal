Relacion entre matematicas y programacion

Comenzaremos con una comparacion entre los tipos de variables numericas en programación y como se relacionan con conceptos matemáticos basicos.

1. Tipos de datos numericos

  • Enteros (int): Numeros sin decimales. Ejemplo: 5, -3
  • Flotantes (float): Numeros con decimales. Ejemplo: 3.14, -0.5
  • Booleanos: Aunque no son numéricos, pueden representar 1 (verdadero) y 0 (falso)

2. Operaciones basicas

En la mayoría de lenguajes de programación, puedes realizar operaciones como:

  • + → Suma
  • - → Resta
  • * → Multiplicación
  • / → División
  • % → Módulo
